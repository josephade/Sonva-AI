# Sonva AI – Frontend Web Application
*AI Receptionist for Modern Clinics*

Sonva AI is an intelligent virtual receptionist that answers calls, books appointments, handles patient queries, and operates 24/7. This repository contains the **Next.js 15 frontend website** for Sonva AI, focused on high performance, modern design, and conversion-driven UX.

---

## 🚀 Tech Stack
- **Next.js 15.3.0**
- **React 18 (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide Icons**
- **Supabase (optional integrations)**
- **Vercel Deployment**
